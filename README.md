# Recipes App By Rohan Puri
## Features

- Recipes List
- Categories
- Ingredients
- Recipes Details Screen
- Detailed Description
- Search
- Ingredients List
- Recipes by Ingredient
- Search
- Used Stack Menu
- Navigation Bar
- Beautiful UI Design Kit
- Compatible with Expo
- Highly modularized codebase
